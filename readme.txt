Works with the Formidable plugin to send SMS notifications when somebody 
submits your contact form, using the Clockwork API. 

== Description ==

Adds an SMS box to your Formidable options pages, fill this in and you'll 
get a text message each time somebody fills out one of your forms.

You need a [Clockwork SMS account](http://www.clockworksms.com/platforms/wordpress/?utm_source=wordpress&utm_medium=plugin&utm_campaign=formidable) and some Clockwork credit to use this plugin.

= Requires =

* Wordpress 3 or higher

* [Formidable Pro](http://formidablepro.com)

* A [Clockwork SMS account](http://www.clockworksms.com/platforms/wordpress/?utm_source=wordpress&utm_medium=plugin&utm_campaign=formidable)

== Installation ==

1. Upload the 'formidable-sms' directory to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Enter your Clockwork API key in the 'Clockwork Options' page under 'Clockwork SMS'
4. Set your SMS options for each Formidable Form

== Frequently Asked Questions ==

= What is a Clockwork API key? = 

To send SMS you will need to sign up for a Clockwork SMS account
and purchase some SMS credit. When you sign up you'll be given an API key.

= Can I send to multiple mobile numbers? =

Yes, separate each mobile number with a comma.

= What format should the mobile number be in? =

All mobile numbers should be entered in international format without a 
leading + symbol or international dialing prefix.  

For example a UK number should be entered 447123456789, and a Republic 
of Ireland number would be entered 353870123456.
